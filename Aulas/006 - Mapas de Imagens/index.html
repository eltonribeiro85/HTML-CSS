Mapas De Imagens

A tag HTML <map> define um mapa de imagem.
Um mapa de imagem é uma imagem com áreas clicáveis.
As áreas são definidas com uma ou mais <area> tags.

Forma

Você deve definir o formato da área clicável e pode escolher um destes valores:

  • rect - Define uma região retangular
  • circle - Define uma região circular
  • poly - Defie uma região poligonal
  • default - Define toda a região
